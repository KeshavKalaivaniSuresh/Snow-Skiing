2D Snowboarder Game (Physics-Based Mini Game)
A 2D mini-game where players control a snowboarder navigating wavy snow terrain from start to finish.

Features:
- Physics-based movement (gravity, momentum, slope interaction)
- Collision detection with crash and reset mechanics

Particle system:
- Color effects on completion
- Explosive effects on crashes

Technologies Used:
- Unity 
- C#

Concepts Applied:
- Game Physics
- Particle Systems
- Performance Optimization
- Object-Oriented Programming

How to Run:
Open the project in the game engine
Run the main scene to start the game
